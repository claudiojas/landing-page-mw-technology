Bem-vindo ao repositório oficial da **Módulo Web Technology**. Este projeto é uma vitrine de alta performance e engenharia aplicada ao marketing digital, focada em converter visitantes em leads de alto valor através de experiências ultrarrápidas e inteligência artificial.

## 🚀 Diferenciais de Elite

Este projeto não é um "site comum". Ele foi construído com os mais modernos padrões de desenvolvimento para garantir resultados reais de ROI para os clientes:

- **Performance Extrema**: Otimização total para atingir 100/100 no Google Lighthouse.
- **Engenharia de Vendas**: Substituímos formulários estáticos por um **Diagnóstico de Performance de Elite** interativo.
- **Design Neo-Clinical**: Uma estética futurista, limpa e premium que transmite autoridade imediata.
- **IA Lead Scoring**: Preparado para integração com IA para análise técnica e de mercado em tempo real.

## 🛠️ Stack Tecnológica

- **Core**: Next.js 16 (App Router) & React 19.
- **Estilização**: Tailwind CSS 4.0 (Engine de alto desempenho).
- **Animações**: Framer Motion para micro-interações fluidas.
- **UI Components**: Radix UI & Shadcn/UI customizado para a Módulo Web.
- **Tipografia**: Otimização de fontes via Google Fonts (Inter/Outfit).
- **Ícones**: Lucide React.

## 🏗️ Arquitetura do Projeto

Seguimos uma estrutura resiliente e escalável baseada em padrões de **Design Atômico** e **Atomic Architecture**:

- `app/`: Roteamento e layouts globais.
- `app/diagnostics/`: Rota exclusiva de conversão (Squeeze Page).
- `components/landing/`: Componentes específicos da Landing Page (Hero, Diferenciais, Footer, etc).
- `components/ui/`: Base de componentes visuais fundamentais.
- `app/assets/`: Ativos de marca e imagens de performance.

## 🏁 Como Começar

### Pré-requisitos
- Node.js 20+
- npm ou pnpm

### Instalação
```bash
# Clone e entre no diretório
git clone [repository-url]
cd [directory-name]

# Instale as dependências
npm install

# Inicie o modo desenvolvimento
npm run dev
```

### Build para Produção
```bash
npm run build
npm run start
```

## 📈 Lead Generation: Diagnóstico de Elite

A principal ferramenta de conversão deste projeto encontra-se em `/diagnostics`.
- **Isolated Squeeze Page**: Sem navegação externa para foco total no formulário.
